## Delanator 
### ломается сетка при >400k точек
Отрисовка границы сетки в RenderLib.TaskRendererFields.RenderBoundary(). <br>
Отрисовка выполнениется в цикле полинейно, т.е. для квадрата он нарисует 4 линии

## Triangle Net
### поломка smooth
1. Если не все точки, входят в ограниченную область, то вылезает исключение (пример: Pem.poly.): <br>
TriangleNet.Smoothing.SimpleSmoother.Centroid(). edge == null <br>
System.NullReferenceException: "Object reference not set to an instance of an object." <br>
edge - атрибут класса Face. Вызов метода Centroid() происходит из метода Step() того же класса, который в свою очередь прогоняется в цикле сглаживания (Smooth). <br>
Причем атрибут edge у Face = null на первой же итерации.
2. Функция smooth не работает из-за исключения в методе Step(): face == null. <br>
В данном случае (возможно) проблема происходит с последний элементом face списка Faces.
